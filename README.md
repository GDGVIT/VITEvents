## Functionalities
- [X]  Get all the events from mail ID
- [X]  Allow adding of events to calendar
- [X]  Make changes for compability to new online format

<br>


## Instructions to run

* Pre-requisites:
	- Flutter
	- Firebase Web API Key
	- OCR API Key
	- Setup .env with WEB_API_KEY and OCR_API_KEY

* Installation:
```bash
flutter pub get
```

* Execution

```bash
flutter run
```
